# foxglove-edit-params version history

## 1.0.0

- Initial public release
- View and adjust ROS 2 parameters from a dedicated Foxglove panel
- Supports boolean, integer, double, and string parameter types
- Multiple input controls: sliders, checkboxes, numeric inputs, and text fields
- Persistent node/parameter selection between panel sessions
- Real-time parameter updates via the Foxglove bridge (no separate rosbridge required)
- Multiple panels can be used simultaneously and arranged freely
